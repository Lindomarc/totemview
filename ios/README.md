# Totem View iOS

Este diretório contém o código-fonte para a versão iOS do app.

## Como usar

1. Abra o Xcode em um Mac.
2. Crie um novo projeto iOS com SwiftUI e nomeie como `Totem View`.
3. Substitua o conteúdo de `AppNameApp.swift`, `ContentView.swift` e adicione `WebView.swift` a partir deste diretório.
4. Ajuste o bundle identifier e as configurações de assinatura se necessário.
5. Defina o `@main` app file como `AppBrowserApp.swift`.

- O app abre a página em um WebView em tela cheia.
- A interface é SwiftUI para iOS 15+.
- Para gerar o app final no iPhone, use Xcode.
